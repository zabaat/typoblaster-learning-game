# Content, Tone, and Naming

## Game Name
TypoBlaster

## Tagline Options
Primary:
- Every key is ammo.

Alternates:
- Type fast. Blast faster.
- The keyboard fights back.
- A tiny typing arcade.

## Theme
Keyboard Wizard defending the realm from letter goblins.

## Tone
Playful, old-school, slightly absurd.

## Avoid
- Corporate language
- Educational/testing language
- Overexplaining
- Modern app jargon
